(Operators)

(There are different types of javascript operators:)

  • Arithmetic Operator
  • Assignment Operator
  • Comparison Operator
  • logical Operator

  • Arithmetic Operator

(Arithmetic operators perform arithmetic on numbers (literals or variables).)

  1. Addition Operator (+)
  2. Subtraction Operator (-)
  3. Multiplication (*)
  4. Exponentiation (**)
  5. Division Operator (/)
  6. Modulus Operator (%)
  7. Increment Operator (++)
  8. Decrement Operator (--)

1.Addition Operator (+)

(It is used to add two or more variables.)

let a = 5;
let b = 5;
let c = a + b;

Output

2.Subtraction Operator (-)

(is is used to subtract two or more variables.)

let d = 5;
let e = 5;
let f = d + e;

Output

3.Multiplication Operator (*)

(Is is used to multiply two or more variables.)

let g = 5;
let h = 5;
let i = g * h;

Output

4.Exponentiation Operator (**)

(It is used to take the exponential value form two or more variables.)

let j = 5;
let k = 5;
let l = j ** k;

Output

5.Division Operator (/)

(It is used to divide two or more variables.)

let m = 5;
let n = 5;
let o = m / n;

Output

6.Modulus Operator (%)

(It is used to get reminder value from two values.)

let p = 5;
let q = 5;
let r = p % q;

Output

7.Increment Operator(++)

(It is used to increase a value of variable by 1.)

let s = 5;
s++;

Output

8.Decrement Operator(--)

(It is used to decrease the value of a variable by 1.)

let t = 5;
t--;

Output

  • Assignment Operator

(Assignment operators assign values to JavaScript variables.)

  1. Equal Operator (=)
  2. Addition Equal Operator (+ =)
  3. Subtraction Equal Operator (- =)
  4. Multiplication equal Operator (* =)
  5. Division Equal Operator (/ =)
  6. Modulus Equal Operator (% =)
  7. Exponentiation Equal Operator (** =)

1.Equal Operator (=)

(It is used to assign values to variables)

let u = 5;

Output

2.Addition Equal Operator (+ =)

(It is used to assign and add a value)

let v = 5;
v += 5;

Output

3.Subtraction Equal Operator (- =)

(It is used to assign and subtract a value)

let w = 5;
w -= 5;

Output

4.Multiplication Equal Operator (* =)

(It is used to assign and multiply a value)

let x = 5;
x *= 5;

Output

5.Division Equal Operator (/ =)

(Is is used to assign and divide a value.)

let y = 5;
y /= 5;

Output

6.Modulus Equal Operator (% =)

(It is used to assign a value and get reminder forma value)

let z = 5;
z %= 5;

Output

7.Exponentiation Equal Operator (** =)

(It is used to assign a value and take the exponentiation of a value.)

let _a = 5;
_a **= 5;

Output

  • Comparison Operator

(Comparison operators are used to compare two values).

  1. Equal to Operator (= =)
  2. Equal value and Equal type (= = =)
  3. Not Equal Operator (! =)
  4. Not Equal value and Not Equal type (! = =)
  5. Greater than Operator (>)
  6. Less than Operator (<)
  7. Greater than or Equal to Operator (> =)
  8. Less than or Equal to Operator (< =)

1.Equal to Operator (= =)

(It is used to compare two values are equal!)

let _b = 5;
_b == 5;

Output

2.Equal value and Equal type (= = =)

(It is used to compare two values are equal!)

let _c = 5;
_c === 5;

Output

3.Not Equal Operator (! =)

(It is used to compare two values)

let _d = 5;
_d != 5;

Output

4.Not Equal value and Not Equal Type operator (! = =)

(It is used to compare two values)

let _e = 5;
_e !== 5;

Output

5.Greater than Operator (>)

(It is used to compare two values)

let _f = 5;
_f !== 5;

Output

6.Less than Operator (<)

It is used to compare two values

let _g = 5;
_g < 3;

Output

7.Greater than or Equal to Operator (> =)

It is used to compare two values

let _h = 5;
_h >= 3;

Output

8.Less than or equal to operator (< =)

It is used to compare two values

let _i = 5;
_i <= 3;

Output

  • Logical Operator

(Logical operators are used to determine the logic between variables or values.)

  1. And Operator (& &)
  2. OR Operator (| |)
  3. Note Operator (!)

1.And Operator (& &)

(The OR (&&) operator returns true if both expressions are true, otherwise false:)

let _j = 5;
let _k = 5;
(_j < 10 & & _k > 1)

Output

2.OR Operator (| |)

(The OR (| |) operator returns true if one or both expressions are true, otherwise false:)

let _l = 5;
let _m = 5;
(_l === 5 || _m === 5)

Output

3.Not Operator (!)

(It is used to revers the value)

let _n = 5;
!(_n == 8);

Output
